Q1.
            function counter(){
                var counter = 0;
            
                function IncreaseCounter() {
                    return counter += 1;
                };
            
                return IncreaseCounter;
            }
            
            var counter = counter();
            alert(counter());
            alert(counter());
            alert(counter());
            alert(counter());

Ans. alert message -- 1 / alert message -- 2 / alert message -- 3 / alert message -- 4
Explanation:- In a function counter value Inizialized with 0. function under function inc. the value of counter by 1 and counter function call 4 times thats why we got counter inc.d value.
